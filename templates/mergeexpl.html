Merge Sort


Explanation


Sorting algorithms are used to take an unordered list (3,2,5,1,4), and output an ordered list containing the same elements (1,2,3,4,5). Merge sort is a more complex, yet very efficient sorting algorithm. Merge sort is made up of two main steps, splitting and merging. The splitting phase involves halving the list into two sub-lists repeatedly, until only single elements remain. The merging phase involves taking these sub lists, and combining them into larger ordered sub lists, until a single, fully sorted list remains.
